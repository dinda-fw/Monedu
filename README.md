# MONEDU

Platform edukasi literasi keuangan interaktif dengan gamifikasi untuk meningkatkan pengetahuan dan keterampilan keuangan masyarakat Indonesia.

## ✨ Fitur Utama

- **📚 Pustaka Materi Finansial**: Koleksi modul pembelajaran komprehensif tentang berbagai aspek keuangan
- **🎯 Kuis Interaktif**: Uji pengetahuan dengan kuis yang menarik dan informatif
- **🏆 Tantangan**: Tantangan harian dan mingguan untuk meningkatkan keterampilan
- **📊 Papan Peringkat**: Kompetisi dengan pengguna lain dan lihat peringkatmu
- **🤖 Tutor AI**: Asisten virtual untuk bantuan pembelajaran
- **🎮 Gamifikasi**: Sistem poin, XP, dan koin untuk motivasi belajar
- **📱 Responsive Design**: Akses di desktop dan mobile dengan pengalaman optimal
- **🌙 Dark Mode**: Dukungan tema gelap dan terang

## 🚀 Teknologi

- **Framework**: Next.js 16
- **Bahasa**: TypeScript
- **Styling**: Tailwind CSS
- **UI Components**: Radix UI
- **Animasi**: Framer Motion
- **3D Graphics**: Three.js, React Three Fiber
- **Form Handling**: React Hook Form, Zod
- **Charts**: Recharts
- **State Management**: SWR
- **Analytics**: Vercel Analytics

## 📋 Prasyarat

- Node.js 18+
- pnpm (Package Manager)

## 🛠️ Instalasi

1. Clone repository ini:
```bash
git clone https://github.com/dinda-fw/monedu.git
cd monedu
```

2. Install dependencies:
```bash
pnpm install
```

3. Jalankan development server:
```bash
pnpm dev
```

4. Buka [http://localhost:3000](http://localhost:3000) di browser Anda.

## 📜 Scripts

- `pnpm dev` - Jalankan development server
- `pnpm build` - Build untuk production
- `pnpm start` - Jalankan production server
- `pnpm lint` - Jalankan ESLint

## 📁 Struktur Proyek

```
monedu/
├── app/                    # Next.js App Router
│   ├── (app)/             # Protected routes
│   ├── (auth)/            # Authentication routes
│   └── globals.css        # Global styles
├── components/            # Reusable components
│   ├── ui/               # UI components (Radix UI)
│   └── 3d/               # 3D components
├── hooks/                # Custom React hooks
├── lib/                  # Utility libraries
│   ├── auth-context.tsx  # Authentication context
│   ├── game-data.ts      # Game data and logic
│   └── utils.ts          # Utility functions
└── public/               # Static assets
```
